# solar java test 2026-02-01

Консольное приложение, которое управляет таблицей из двух колонок <имя, дата> содержащей список дней рождения.

Таблица хранится в файле формата SQLite, приложение имеет следующий функционал представленный меню ниже.

        Выберите действие:
        1. Все записи
        2. Ближайшие дни рождения     
        3. Добавить день рождения     
        4. Удалить день рождения      
        5. Редактировать день рождения
        6. Выход
        >
